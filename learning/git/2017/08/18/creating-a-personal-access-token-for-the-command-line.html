A personal access token

Lee Murray, 18 August 2017

I use iterm 2 for my git activities, it just feels right, I also use two factor authentication which you should be using too.

Create a token so you don’t need to enter a password every time your performing git operations over https.


git clone https://github.com/username/repo.git
Username: your_username
Password: your_token

For info on how to create a personal access token, have a look here.
