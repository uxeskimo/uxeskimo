CSS target first letter

Lee Murray, 15 August 2017

Here’s something I didn’t know you could specificly target via CSS, the first letter in a element.

p::first-letter {
  font-size: 40px;
  color: #21d67f;
}

Which would give you

This is an example line of text.

::first-letter is a pseudo element which works only if the parent element is a block container.
