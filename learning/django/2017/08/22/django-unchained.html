Django unchained

Lee Murray, 22 August 2017

Today to help the team I put my developer hat on (designers wear many hats you know) to have a look at Django which is an open source Python web framework.

Django encourages quick development and has ‘out of the box’ features for you to use on your project, like a ‘login system’ for example (more benefits here).

My task is to get some Django templates set up so we can add our design in HTML and CSS.

 I was told about a decent resource https://tutorial.djangogirls.org/en/django/ which had step by step instructions to get going with Django, first off Python needs to be installed which I downloaded from https://www.python.org/downloads/release/python-351/.

To check it installed OK, I opened up iterm2 and entered

$ python3 --version

This told me I had Python 3.5.1 installed so was good to go, next up I then made a virtualenv which is an isolated Python environment (virtual environment), I did this with

$ python3 -m venv myvenv

and to get it started I used

$ source myvenv/bin/activate

with dependancies set up I could then install Django

$ pip install django

I’m now set up to create the design templates we need (I hope).
